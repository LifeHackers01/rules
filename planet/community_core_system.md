#### 0.2 信息分级

| 类别 | 代号 | 等级 | 点赞数 | 正面评论数 | 两数关系 |
|:--|:--|:--|:--|:--|:--|
| 消息 | MSG | A | [SUM x 20%, ] | [SUM x 10%, ] | 并且 |
| 消息 | MSG | B | [SUM x 10%, SUM x 20%) | [SUM x 5%, SUM x 10%) | 并且 |
| 消息 | MSG | C | [SUM x 1%, SUM x 10%) | [SUM x 1%, SUM x 5%) | 或者 |
| 消息 | MSG | D | [1, SUM x 1%) | [1, SUM x 1%) | 或者 |
| 消息 | MSG | E | 0 | 0 | 或者 |
| 文章 | ATC | A | [SUM x 20%, ] | [SUM x 10%, ] | 并且 |
| 文章 | ATC | B | [SUM x 10%, SUM x 20%) | [SUM x 5%, SUM x 10%) | 并且 |
| 文章 | ATC | C | [SUM x 1%, SUM x 10%) | [SUM x 1%, SUM x 5%) | 或者 |
| 文章 | ATC | D | [1, SUM x 1%) | [1, SUM x 1%) | 或者 |
| 文章 | ATC | E | 0 | 0 | 或者 |
| 评论 | CMT | A | [SUM x 7%, ] | [SUM x 3%, ] | 并且
| 评论 | CMT | B | [1, SUM x 7%) | [1, SUM x 3%) | 或者
| 评论 | CMT | C | 0 | 0 | 或者

**注：**

1. 信息：社群内发表的消息/主题、文章和评论的统称。
2. 正面评论：非违规的、非恶意的，且不包含反对、声讨、谴责、抨击等内容的积极评论。
3. 违规内容：涉嫌违反国家、地区、组织、团体的各项法律、法规的内容。形式包括且不限于文字、图片、视频等。
4. 恶意内容：涉及国家、民族、阶级、地域、氏族等仇恨的内容，以及带有三俗、谩骂、侮辱性质的内容。形式包括且不限于文字、图片、视频等。
5. 两数关系：达到对应等级所需的点赞数和正面评论数之间的关系，可选值有“并且”和“或者”。
6. SUM：社群成员总数，并四舍五入向百位取整。例如：社群成员总数为150人，向百位取整得到200，即：SUM=200。

### 1. 基本规则

1. 在本社群中，任何成员都有机会晋级为嘉宾甚至合伙人。
2. 本社群中的所有嘉宾都可以（且仅可以）自由发表与社群主题强相关的信息，并有机会获得奖励。
3. 本社群中的所有合伙人都拥有嘉宾的权利和义务，并可以在达准后获得分红。

### 2. 成员晋级规则

#### 2.1 从群众到嘉宾

1. 群众可积极参与本社群内的信息的讨论，次数不限。
2. 一旦某位群众发表的某个评论达到A级，他就可以得到`1`枚社币。
4. 群众需要自行以**屏幕截图**的形式上报社币增量给社群的管理员。社群管理员只会记录群众已上报的社币增量。
5. 上报需要具有时效性。若在上报时相关信息的发表时间已在`10`天以前，那么该次上报将会被视为无效。
6. 一旦某位群众的社币满`50`枚，就会得到`1`张晋级令牌。
7. 管理员会在每月的月初进行嘉宾提名，即：分别公布社币最高的`5`名群众。全体成员可通过点赞的方式选出`1`名群众，得到点赞数量最多者将胜出。
8. 如果届时嘉宾数量未达到上限，那么胜出的群众会立即晋级为嘉宾。否则，该名群众会暂时被列为预备嘉宾，待有空位时再晋级。

#### 2.2 从嘉宾到合伙人

1. 嘉宾可以（且仅可以）自由发表与社群主题强相关的信息。
2. 一旦某位嘉宾发表的某个消息或某篇文章达到了A级，他就可以得到`2`枚社币。
3. 一旦某位嘉宾发表的某个消息或某篇文章达到了B级，他就可以得到`1`枚社币。
4. 嘉宾需要自行以**屏幕截图**的形式上报社币增量给社群的管理员。社群管理员只会记录嘉宾已上报的社币增量。
5. 上报需要具有时效性。若在上报时相关信息的发表时间已在`10`天以前，那么该次上报将会被视为无效。
6. 一旦某位嘉宾的社币满`100`枚，就会得到`1`张晋级令牌。前提是嘉宾在当季不得有`1`块及以上的社砖，否则无法获得晋级令牌。详见“成员降级规则”。
7. 管理员会在每个季度的季末（每年的1月、4月、7月、11月的月末）进行合伙人提名，即：分别公布社币最高的`3`名嘉宾。全体成员可通过点赞的方式选出`1`名嘉宾，得到点赞次数最多者将胜出。
8. 如果届时合伙人数量未达到上限，那么胜出的嘉宾会立即晋级为合伙人。否则，该名嘉宾会暂时被列为预备合伙人，待有空位时再晋级。

### 3. 成员降级规则

#### 3.1 对于全体成员

所有成员不得发表不当言论，包括：

- 违规内容
- 恶意内容

**一经举报或发现，管理员会立即向全体成员发布公告，并将相关成员请出本社群！**

#### 3.2 对于嘉宾

- 嘉宾发表消息和文章的总频次不得小于每周`2`次，否则将会得到`1`块社砖。
- 嘉宾需要回答社群成员们提出的问题。若在一个月内有超过`3`位成员向管理员抱怨某位嘉宾，一经查实，该嘉宾就将得到`1`块社砖。
- 若单季度内得到`2`块及以上的社砖，则该嘉宾将被降级。若此时有未使用的晋级令牌，则收回该令牌而不降级。

#### 3.3 对于合伙人

- 合伙人发表消息和文章的总频次不得小于每周`3`次，否则将会得到`1`块社砖。
- 合伙人需要回答成员们提出的问题。若在一个月内超过`3`位成员向管理员抱怨某位合伙人，一经查实，该合伙人就将得到`1`块社砖。
- 若单季度内得到`3`块及以上的社砖，则该合伙人将被降级。

### 4. 成员奖励规则

#### 4.1 对于全体成员：

本社群会不定期布置作业题，并根据成员“写作业”的质量发放奖励。具体规则如下：

- 对于同一道作业题的回答，根据点赞数进行排名。
- 每一道作业题都只奖励前三名。
- 第一名将获得奖励`20`元整，第二名将获得奖励`10`元整，第三名将获得奖励`5`元整。
- 作业回答不得少于`200`字的有效内容，否则不计入名次。

#### 4.2 对于嘉宾：

本社群会把上个月净收入的`20%`划入嘉宾奖励池，并专门用于奖励嘉宾发表的优质内容。具体规则如下：

- 嘉宾在前一个月发表的消息和文章的总数量不得少于`10`个，否则无法获得奖励。
- 一旦某位嘉宾在前一个月发表的消息或文章达到`[2, 5)`个A级，他就将得到奖励`30`元整。
- 一旦某位嘉宾在前一个月发表的消息或文章达到`[5, )`个A级，他就将得到奖励`50`元整。

**注意：**

1. 前一个月的嘉宾奖励会在当月的月初统一发放。
2. 奖励池每月自动清空，即：当月之前的奖金不会进入下个月的奖励池。
3. 按照嘉宾们达标的时间顺序发放奖金，直到当月的奖励池已空为止。

#### 4.3 对于合伙人：

原则上，前`2`名的合伙人在达标后都可以获得分红。具体金额是，每人每季度获得本社群在当季度的净收入的`15%`。但前提如下：

- 合伙人在当季不得有`2`块及以上的社砖，否则无法获得奖励。
- 合伙人在当季中的每个月发表的消息和文章的总数量均不得少于`20`个，否则无法获得奖励。
- 合伙人在每个月都至少要有`5`个消息或文章达到`B`级或更好，否则无法获得奖励。

**管理员将会以嘉宾发表的消息和文章的等级的数量作为排名依据。**
